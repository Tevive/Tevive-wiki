# Contact

Thank you for your interest in Tevive.

## General Questions

For general questions, please open a GitHub Discussion or Issue.

## Bug Reports

If you discover a bug, please create a Bug Report issue.

## Feature Requests

If you have an idea for a new feature, please create a Feature Request issue.

## Game Requests

To request a game, please use the Game Request template.

## Business & Legal

For business inquiries, partnerships, copyright concerns, or legal matters, please contact the project owner through GitHub.

## Community

Community feedback helps improve Tevive and shape future updates.

Thank you for supporting Tevive.
