# Installation

## Requirements

Before installing Tevive, make sure your system meets the following requirements:

* Windows 10 or newer
* Internet connection
* Latest available Tevive release

## Download

1. Go to the Releases section.
2. Download the latest version of Tevive.
3. Extract the files if necessary.

## Installation

1. Run the installer or executable.
2. Follow the on-screen instructions.
3. Launch Tevive.

## Updating

To update Tevive:

1. Download the latest release.
2. Replace your existing installation.
3. Launch the updated version.

## Troubleshooting

If you encounter issues:

* Restart the application.
* Download the latest version.
* Check the GitHub Issues page.
* Contact the Tevive team.
